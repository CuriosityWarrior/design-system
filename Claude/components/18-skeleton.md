# 컴포넌트: 스켈레톤 (Skeleton)

## 개요
콘텐츠 로딩 중 표시되는 플레이스홀더 UI. 레이아웃 이동 방지.

---

## 디자인 토큰

| 토큰 | 값 |
|---|---|
| 기본 색상 | `color/surface/3` |
| 시머 색상 | `color/border/default` |
| 애니메이션 | shimmer 1.5s infinite (배경 위치 스윕) |
| 텍스트 반경 | `radius/S` (6px) |
| 원형 반경 | 50% |
| 사각형 반경 | `radius/card` = 12px |

---

> 📐 사이즈 기본 원칙은 [파운데이션: 사이즈](../foundations/07-size.md) 참조. 스켈레톤은 대체 대상 컴포넌트의 사이즈 토큰을 따른다 (예: 원형 스켈레톤 → `size/avatar/*`).

## 요소 종류

### 텍스트 스켈레톤
- 높이: 12px (S), 14px (기본), 18px (L)
- 보더 반경: `radius/S` (6px)
- 너비: 다양 (100%, 85%, 65% 등)
- 하단 여백: 줄 사이 8px

### 원형 스켈레톤
- 아바타 자리 표시자로 사용
- 보더 반경: 50%
- 일반 크기: 36px, 44px

### 사각형 스켈레톤
- 이미지, 카드, 미디어 자리 표시자
- 보더 반경: `radius/card` (12px)
- 가변 가로/세로

---

## 패턴

### 카드 스켈레톤
- 헤더 행: 원형 (아바타) + 텍스트 두 줄
- 대형 사각형 (이미지/미디어 영역)
- 하단 텍스트 세 줄

### 목록 스켈레톤
- 반복 행: 원형 + 텍스트 두 줄 + 짧은 우측 텍스트

---

## 사이즈 동작

| 속성 | 값 |
|---|---|
| layoutSizingHorizontal | `FIXED` 또는 `FILL` (실제 콘텐츠 형태에 맞춤) |
| layoutSizingVertical | `FIXED` (실제 콘텐츠 형태에 맞춤) |

> 스켈레톤은 로딩 후 대체될 실제 콘텐츠(텍스트 줄, 이미지 등)의 크기를 모사하므로 해당 컨텍스트에 맞는 크기를 지정한다. HUG 사용 금지(자식 요소가 없기 때문).

---

## 아이콘 사용 규칙

> 컴포넌트 내 아이콘은 반드시 `01 — Icons` 페이지의 아이콘 컴포넌트 인스턴스를 사용한다.
> 텍스트 특수 문자(✓, ✕, →, ⋯ 등), 이모지, 직접 그린 벡터 도형으로 아이콘을 대체하는 것을 금지한다.
> 필요한 아이콘이 없는 경우, 먼저 `01 — Icons` 페이지에 추가한 후 인스턴스를 참조한다.

---

### Variants 구성
- 모든 변형은 Figma의 **Combine as Variants** 기능을 사용하여 하나의 Component Set으로 통합한다.

## 사용 원칙

| 원칙 | 설명 |
|---|---|
| 실제 콘텐츠 형태 모사 | 스켈레톤의 크기와 형태는 로딩 후 표시될 실제 콘텐츠(텍스트 줄 수, 이미지 비율, 아바타 크기)를 최대한 닮아야 한다. |
| 로딩 중에만 표시 | 스켈레톤은 데이터 로딩 중에만 노출하고, 데이터 수신 즉시 실제 콘텐츠로 교체한다. 로딩 완료 후에도 스켈레톤이 남아 있지 않도록 한다. |
| 레이아웃 이동 방지 | 스켈레톤이 실제 콘텐츠와 동일한 크기를 점유해 콘텐츠 로딩 후 레이아웃이 재배치(CLS)되지 않도록 한다. |
| Spinner와 중복 사용 금지 | 같은 영역에 스켈레톤과 Spinner를 동시에 사용하지 않는다. 전체 페이지 로딩엔 스켈레톤, 버튼 내 인라인 로딩엔 Spinner를 사용한다. |
| 과도한 디테일 지양 | 스켈레톤은 대략적인 레이아웃을 암시하는 수준으로만 표현한다. 실제 콘텐츠의 모든 세부 요소를 스켈레톤으로 재현하려 하지 않는다. |

## Figma Make 프롬프트

```
다음 스펙으로 스켈레톤(Skeleton) 로딩 플레이스홀더 컴포넌트를 만들어줘:

기본 스타일: 시머 애니메이션이 있는 연한 회색 채움 (좌→우 스윕)

요소:
- 텍스트 스켈레톤: 둥근 사각형, 높이 12px/14px/18px
- 원형 스켈레톤: 완전한 원형 (아바타 자리 표시자)
- 사각형 스켈레톤: 둥근 사각형 (이미지/카드 자리 표시자)

패턴:
- 카드 스켈레톤: 아바타 행 + 이미지 영역 + 텍스트 3줄
- 목록 스켈레톤: 아바타 + 텍스트 2줄 + 짧은 우측 텍스트로 구성된 3개 행

모든 요소에 동일한 회색 시머 애니메이션 적용

네이밍: Skeleton / Text / {크기}, Skeleton / Card, Skeleton / List
```
